Ce projet Python utilise \*\*GitHub Actions\*\* pour l'intégration continue. Il inclut des tests unitaires, du linting, et un suivi de la couverture de code.



\## Fonctionnalités



\- \*\*Tests automatisés\*\* : Utilisation de `pytest` pour exécuter les tests.

\- \*\*Linting\*\* : Vérification du style de code avec `flake8`.

\- \*\*Couverture de tests\*\* : Mesure du pourcentage de couverture du code via `pytest-cov`.
